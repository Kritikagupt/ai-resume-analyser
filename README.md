# AI Resume Analyzer 🤖📄

An AI-powered web application that analyzes resumes and provides intelligent feedback using **Google Gemini AI**.

The system allows users to upload their resume and receive AI-generated suggestions to improve their job applications.
This project demonstrates **Natural Language Processing (NLP), Generative AI integration, and file processing using Python.**

---

## 🚀 Features

* Upload resume in **PDF or TXT format**
* Extract text from resume files
* AI-powered resume analysis using **Gemini API**
* Instant feedback to improve resume quality
* Simple web interface built with **Streamlit**

---

## 🛠 Technologies Used

* Python
* Streamlit
* Google Gemini API
* PyPDF2
* Python-dotenv
* Natural Language Processing (NLP)

---

## ⚙️ How to Run the Project

### 1. Clone the repository

git clone https://github.com/Kritikagupt/ai-resume-analyser.git

### 2. Navigate to the project folder

cd ai-project

### 3. Install dependencies

pip install -r requirements.txt

### 4. Run the application

streamlit run main.py

---

## 🔄 Project Workflow

1. User uploads a resume file.
2. The system extracts text from the document.
3. The extracted text is sent to **Google Gemini AI**.
4. AI analyzes the resume and provides feedback.
